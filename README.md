#Explore Egypt – Travel Recommendation Web Application

Explore Egypt is a modern and interactive travel recommendation web application built using HTML, CSS, and JavaScript. The platform allows users to discover amazing destinations in Egypt through simple keyword-based searches.

✨ Features
🧭 Multi-page structure (Home, About Us, Contact Us)
🔎 Smart search functionality for destinations (beaches, temples, cities, etc.)
📄 Dynamic recommendations loaded from a JSON file
⚡ Real-time filtering and result display
🔄 Reset search functionality for better user experience
📱 Responsive and user-friendly design
🏜️ Homepage Experience
Stunning hero section with an Egypt-inspired background
Engaging headline encouraging exploration of Egypt’s beauty
Short cultural and travel-focused description
Social media integration
“Book Now” button for user engagement
🛠️ Technologies Used
HTML5
CSS3
JavaScript (ES6)
DOM Manipulation
Fetch API
🎯 Project Goal

This project was built as part of a JavaScript Essentials final assignment to demonstrate the ability to create a fully functional, interactive, and visually appealing travel recommendation web application.
